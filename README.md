# Онлайн-школа

Сайт для дистанционного обучения учеников начальных классов и дошкольников.

- **Версия для учеников и родителей** — яркий дизайн, персонажи, вход по логину/паролю.
- **Версия для учителя** — синие тона, кабинет: ученики, задания, оценки, онлайн-уроки с доской, микрофоном и камерой.

---

## Как выложить проект на GitHub и открыть сайт по ссылке

### 1. Установить Git (если ещё не установлен)

1. Скачайте Git: https://git-scm.com/download/win  
2. Установите с настройками по умолчанию.  
3. Перезапустите терминал (или Cursor).

### 2. Создать репозиторий на GitHub

1. Зайдите на https://github.com и войдите (или зарегистрируйтесь).  
2. Нажмите **«+»** → **«New repository»**.  
3. Укажите имя, например: **onlineschool**.  
4. Репозиторий сделайте **Public**.  
5. **Не** ставьте галочки «Add README» и «Add .gitignore» — они уже есть в проекте.  
6. Нажмите **«Create repository»**.

### 3. Выложить код с компьютера

Откройте терминал в папке проекта (или в Cursor: **Terminal → New Terminal**) и выполните по очереди:

```bash
cd "c:\Users\user\Desktop\onlineschool"

git init
git add .
git commit -m "Первый коммит: сайт онлайн-школы"

git branch -M main
git remote add origin https://github.com/SvetlanIz/onlineschool.git
git push -u origin main
```

Репозиторий: **https://github.com/SvetlanIz/onlineschool**  
При первом `git push` браузер может запросить вход в GitHub — войдите под **SvetlanIz**.

### 4. Включить GitHub Pages (чтобы сайт открывался по ссылке)

1. На GitHub откройте репозиторий **onlineschool**.  
2. Вкладка **Settings** → слева **Pages**.  
3. В блоке **Source** выберите **Deploy from a branch**.  
4. В **Branch** выберите **main** и папку **/ (root)**.  
5. Нажмите **Save**.

Через 1–2 минуты сайт будет доступен по адресу:

**https://svetlaniz.github.io/onlineschool/**

Этой ссылкой можно пользоваться и передавать другим — по ней откроется главная страница выбора «Ученик и родитель» / «Учитель».
